## 你可能感兴趣的功能

* 对象池：多种对象存储由一个组件管理。
* 敌人AI状态机:每个状态的脚本分离，便于开发与差错。
* 背包系统（无UI）
* 武器系统
* 游戏流程管理组件（回合制）
* 使用观察者模式更新UI
